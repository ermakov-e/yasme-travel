# AGENTS.md

## Project Overview

Yasme Travel — frontend приложение для совместной работы с картой.

Пользователи:

- создают группы
- добавляют точки на карту
- прикрепляют фотографии к точкам

Стек:

- React
- TypeScript
- Vite
- Redux Toolkit
- MUI
- styled-components
- React Router

Архитектура:

- Feature-Sliced Design (FSD)

---

## Architecture

Используется Feature-Sliced Design

Слои:

- app
- pages
- widgets
- features
- entities
- shared

Правила:

- ЗАПРЕЩЕНО нарушать зависимости слоев
- ЗАПРЕЩЕНО смешивать бизнес-логику и UI
- Каждый модуль должен находиться в своем слое

Подробнее:

- docs/fsd.md
- docs/architecture.md

## Routing

Используется React Router

- Реализовать защищенные маршруты

Если пользователь не авторизован:
→ редирект на /login

---

## Authentication (MVP)

Используется mock авторизация:

- login + password
- данные захардкожены
- состояние хранится в Redux
- используется localStorage

Поведение:

- при загрузке проверяется localStorage
- если не авторизован → /login
- после логина → доступ к приложению

## UI and Styling

## Dev Commands

Установка зависимостей:
pnpm install

Запуск проекта (development):
pnpm run dev

Сборка:
pnpm run build

Предварительный просмотр сборки:
pnpm run preview

Линтинг:
pnpm run lint

Проверка типов (TypeScript):
tsc --noEmit
